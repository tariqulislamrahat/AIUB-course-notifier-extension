Installation Guide

Installing on Google Chrome

  • Open Google Chrome.
  • Type chrome://extensions/ in the address bar and press Enter.
  • In the top right corner, toggle on "Developer mode".
  • Click the "Load unpacked" button that appears.
  • Navigate to the folder containing your extension files.
  • Select the folder and click "Select Folder" (on Windows) or "Open" (on Mac).
  • The extension should now appear in your list of extensions.

Installing on Microsoft Edge

  • Open Microsoft Edge.
  • Type edge://extensions/ in the address bar and press Enter.
  • In the bottom left corner, toggle on "Developer mode".
  • Click the "Load unpacked" button that appears.
  • Navigate to the folder containing your extension files.
  • Select the folder and click "Select Folder" (on Windows) or "Open" (on Mac).
  • The extension should now appear in your list of extensions.

Verifying the Installation

  • You should see the extension icon in the browser toolbar.
  • Click on the icon to verify that it's working correctly.
  • Navigate to the AIUB portal (https://portal.aiub.edu/) to test the extension's functionality.

Troubleshooting

  • Ensure all files are in the correct location and have the correct names.
  • Check the browser's console for any error messages.
  • Try reloading the extension from the extensions page.
  • Restart your browser and try the installation process again.
Notice: As this is a developer mode extension, you may need to re-enable it each time you restart your browser, depending on your browser settings.
